

VIDEO SYNTAX
*************

<video width="320" height="240" constrols="controls">
	<source src="movie.mp4" type="video/mp4" />
	<source src="movie.ogg" type="video/ogg" />
</video>



//*****************************
 
If want to use a video like YOUTBE use this syntax

<iframe width="560" height="315" src="http://www.youtube.com/embed/....." frameborder="0" allowfullscreen>
</iframe>
