Methods

a.sort() - sorts out the elements inside the array. Either alphabetically if they are strings, or in ascending order if they are numbers.
a.push() - pushes a new element into the array
a.pop() - removes the last element and returns it
a.join() - adds a string between each element and returns the result as a string
a.slice() - method returns a sub-array without modifying the original array
a.splice() - the splice() methos modifies the array:
it removes "a slice" and also adds new elements
The first two parameters are start and end index, the other
parameters are the elements to add to the array to replace the slice
that will be removed
